# jasmine-starter


Quick and dirty `jasmine-starter` project

## Setup

- clone archive
  - `$ git clone https://github.com/mikeerickson/jasmine-starter.git`
- install dependencies
  - `$ npm install` (or if you prefer to sew `yarn`)
- run using specRunner
  - `$ test` or `$ num run test`
- run using karma
  - `$ npm run test:karma`
